¿Cómo agregar hojas de estilos y archivos estáticos?

1) Descargaremos la carpeta materiales.zip del curso, dentro de ella están varios archivos necesarios para el proyecto, como las carpetas img y css. Se debe crear una carpeta public en la raíz y meter las carpetas img y css dentro.

2) Luego en el index.js de la raíz, vamos a definir la carpeta pública (public), se le puede asignar otro nombre en vez de public, pero la convención es usar public:
// requerimos express
import express from 'express';
// importamos router
import router from './routes/index.js'

const app = express();

// Definir puerto
const port = process.env.PORT || 4000;

// Habilitar PUG
app.set('view engine', 'pug');

// definir la carpeta public
app.use(express.static('public')); // <<---------------------

// Agregar Router
app.use('/', router);

// arranca el servidor con el método .listen, le pasa el port y un callback para 
app.listen(port, () => {
     console.log(`El servidor está funcionando en el puerto: ${port}`)
});
