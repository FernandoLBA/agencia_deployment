Instalación de Node y Express.

1) Descargar la versión actual o current de la página.

2) Crear archivo package.json, en consola ir a la carpeta del proyecto y escribir:
npm init
Luego hará varias preguntas:
-package name: dejas el que está o pones cualquiera nombre
-version: deja la que está
-description: pon lo que quieras
-entry point: (index.js)
-test command: dejar así
-git repository: dejar así, a menos que enlaces un repositorio de git
-keywords
-author: tu nombre
-license: (ISC)
-escribir 'yes' si todo está bien, o 'no' para abortar.

3) Luego hay que instalar las dependencias, hay 2 usos para las dependencias: 1. las requeridas en desarrollo, mientras codificamos nuestra app, se crea un script en devdependencies en package.json y 2. las requeridas en producción, cuando ya subimos el proyecto al servidor y esté publicado, se crean en dependencies en el package.json.

4) Para instalarlo, podemos usar:
-Para producción, porque lo requerimos en el servidor donde se publicará el proyecto:
     npm install express
Esto agregará express a las dependencies del archivo package.json y creará el archivo package-lock.json
-Para desarrollo, porque necesitamos usarlo cuando estemos programando la app:
     npm install --save-dev nodemon
Nodemon sirve para detectar los cambios en el código y reiniciará el servidor automáticamente.
