Conectando una base de datos MySQL

1) Luego de instalar MySQL, por ejemplo podemos usar el programa para windows XAMP o WAMP, este instala el cliente de MySQL, apache, filezilla, etc.

2) Instalar MySQL Workbench para manejar a mysql de forma visual.

3) En MySQL workbench, vamos a iniciar sesión, para este caso con root y admin.

4) Luego importaremos un schema o base de datos así:
-Ir a la sección schema del menú izquierdo, dar click derecho y luego en new schema o en el menú superior dar click en el icono de los discos y el signo +, create a new schema in the conected server, colocar el nombre del schema o base de datos y darle click en apply, nuevamente dar click en apply y finish.
-Luego en la carpeta descargada "materiales.zip" está un archivo llamado agenciaviajes.dump.
-Le damos doble click al schema creado y luego vamos al menú superior, server, data import, seleccionamos import from self-contained file, buscamos el archivo, luego abajo en default target schema, seleccionamos el nombre del schema creado, luego damos click en start import.
-Una vez importado, cerrar la pestaña y revisar el schema, deberá aparecer la tabla viajes.

5) Luego abrir la consola para instalar las dependencias de mysql y sequelize(un ORM), escribiendo:
Para producción:
npm install mysql2 sequelize

6) Creamos una carpeta en la raíz llamada config y dentro de config crear un archivo llamado db.js, el cual va a tener la información de la base de datos, escribimos el siguiente código:
import sequelize from 'sequelize';
const db = new sequelize('agenciaviajes', 'root', 'admin', {
     host: 'localhost', // o el ip local
     port: '3306',
     dialect: 'mysql',
     define: {
          timestamps: false
     },
     pool: {
          max: 5,
          min: 0,
          acquire: 30000,
          idle: 10000
     },
     operatorAliases: false
});

export default db;
Se importa sequelize, se crea una const db que contiene una instancia de sequelize, con el nombre de la bd, el usuario, la contraseña, y un objeto con las configuraciones de mysql y sequelize. 
Luego se exporta db.

7) Abrimos el archivo index.js de la raíz, importamos el db y para conectar a la base de datos escribimos el siguiente código:
// requerimos express
import express from 'express';
// importamos router
import router from './routes/index.js'
import db from './config/db.js'; // <------------------------

const app = express();

// Conectar a BD <----------------------------------
db.authenticate()
     .then( () => console.log('Base de datos conectada') )
     .catch( error => console.log(error))

// Definir puerto
const port = process.env.PORT || 4000;

// Habilitar PUG
app.set('view engine', 'pug');

// Obtener el año actual para el copyright
app.use((req, res, next) => {
     const year = new Date();

     res.locals.ActualYear = year.getFullYear();

     res.locals.NombreSitio = 'Agencia de Viajes';

     return next();
})

// Definir la carpeta public
app.use(express.static('public'));

// Agregar Router
app.use('/', router);

// Arranca el servidor con el método .listen, le pasa el port y un callback para 
app.listen(port, () => {
     console.log(`El servidor está funcionando en el puerto: ${port}`)
});
