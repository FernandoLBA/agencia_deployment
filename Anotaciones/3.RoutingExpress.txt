Routing en Express

1) Crear una carpeta routes en la carpeta raíz, y dentro crear un archivo index.js

 Solo podemos tener una instancia de nuestra app, por ejemplo la que creamos en el archivo index.js de la carpeta raíz, sin embargo podemos importar Express en el nuevo index.js de la carpeta routes:
import express from 'express';

2) Definir el router
const router = express.Router();
Des esta manera estamos instanciando a express, pero utilizando su router.

3) Cortamos las url's del index.js de la carpeta raíz y las pegamos en este nuevo index.js de la carpeta routes, y le cambiamos la variable app a las urls por router.

4) Luego esas rutas se asignan al objeto router de express:
import express from 'express';

const router = express.Router();

router.get('/', (req, res) => {
     res.send('Inicio');
});

router.get('/nosotros', (req, res) => {
     res.send('Nosotros');
});

router.get('/viajes', (req, res) => {
     res.send('Viajes');
});

router.get('/testimoniales', (req, res) => {
     res.send('Testimoniales');
});

Luego se exporta
export default router;

5) Al exportarlo ya se puede importar en el index.js de la carpeta raíz.
import router from './routes/index.js';

luego agregar el router en el mismo index de la carpeta raíz:
app.use('/', router);
El verbo .use soporta todos los verbos get, put, delete, post y patch. Y lo que hace es que desde la página principal agregará router y esto agregará a su vez las distintas url's que se definan.
