Crear vistas con PUG

1) Crear una carpeta views en la raíz

2) Crear archivo nosotros.pug

3) Luego en la carpeta routes, en el archivo index.js, vamos a modificar las url's quitando en el res.send y colocar res.render con el nombre de la vista pug que creamos, automáticamente la cargará:
router.get('/nosotros', (req, res) => {
     res.render('nosotros');
});

4) Le pasamos información a la vista
router.get('/nosotros', (req, res) => {
     const viajes = 'Viaje a Alemania'

     res.render('nosotros', {
          viajes
     });
});
Creamos una variable viajes y se la pasamos como 2do parámetro al método render

5) Luego en la vista cargamos la variable usando las siguientes opciones:
-p= variable
-p #{variable}

6) Luego midificamos las demás url's a render y crearemos sus respectivas vistas en la carpeta views.
inicio.pug, nosotros.pug, viajes.pug y testimoniales.pug

7) Le creamos un contenido básico a cada vista, como por ejemplo, un título y el nombre de la vista:
h1 Agencia de Viajes
p Inicio

8) Sin embargo si tenemos 10, 20 o 30 vistas, y nos piden hacer un cambio en el nombre de la empresa, tendríamos que cambiarlo a cada una, pero se puede crear una página maestra para que las otras páginas hereden ciertos atributos de ella.
